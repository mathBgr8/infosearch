Anzati Code of Honor
1058

The Anzati Code of Honor forbid telling the contracts or the kills. Many Anzati were assassins, and were hired to murder anyone. So, the Code of Honor was created to hide this secret information to the public face when they committed a crime. When Tholme asked the Anzati Assassin master Akku Seii about the Dark Jedi Sora Bulq, the Code protected the information. It also happened when Quinlan Vos asked Sajé Tasha, who was hired to kill the former Supreme Chancellor Finis Valorum. *Republic 72 Category:Anzati culture
