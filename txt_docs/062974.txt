End Run (Xiytiar-class)
2169

End Run was a Xiytiar-class heavy transport operated by the Loose Cannon pirates. During a skirmish with the Azzameen family, End Run was used to transport the bacta lure. While pumping the cargo from End Run to the Vasudra storage tanks, the pirate gang sprang their trap. Unfortunately for the gang, the well-armed Azzameens could not be subdued, and actually sent the pirate ships into retreat if not being destroyed. Only End Run remained, still connected to the Vasudra and unable to hyperspace out.
