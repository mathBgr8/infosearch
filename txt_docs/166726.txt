Code Alpha-five-zed-alpha
696

Code Alpha-five-zed-alpha was a code within Security Protocol Five that Luke Skywalker gave to R7-T1 to send to him on Control Channel One in case anyone got too close to Vader's retreat on Coruscant. Unaware to the astromech droid was that it would notify Luke to destroy the fortress, sending it and the E-wing starfighter into the Western Sea. *Before the Storm Category:Codes
