Mandalorian code
2274

}} The Mandalorian code was a code followed by traditional Mandalorian warriors that could be invoked to settle disputes via one-on-one combat. Sabine Wren used the code to challenge Fenn Rau to a duel after the Protector had injured Hera Syndulla in a battle over Concord Dawn. The individual known as "Frog Lady" thought honoring one's word was a part of the Mandalorian code.
