Computerized interface scope
1241

250px|thumb|An E-11 blaster rifle with an interface scope attached. A computerized interface scope, was a military-grade scope, mounted on a blaster. This type of scope could interface with the wielder's helmet, allowing them to see and lock on to a target from a much farther range. An example of this type of scope would be the scope on the E-11 blaster rifle that could interface with the wielder's helmet, giving him/her an edge on the battlefield.
